# CyberGuard-AI

AI-powered cybersecurity threat detection built with Streamlit.

## Features

- XGBoost attack detection
- Isolation Forest anomaly detection
- Streamlit dashboard
- Real-time predictions

## Installation

```bash
git clone https://github.com/olegbidyuk1234-gif/CyberGuard-AI.git
cd CyberGuard-AI
pip install -r requirements.txt
streamlit run app.py
```

## Technologies

- Python
- Streamlit
- XGBoost
- Scikit-learn
- Pandas
- NumPy
